# IFC Repair SaaS Documentation

Welcome to the documentation for the IFC Repair SaaS. 

This engine is a production-grade, serverless-capable API and Frontend dedicated to automatically validating and repairing OpenBIM Industry Foundation Classes (IFC) models using Python, FastAPI, React, and IfcOpenShell.
